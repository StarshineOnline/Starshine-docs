Constructeur.

Paramètres:
id Id dans la base de donnée ou tableau associatif contenant les informations permettant la création de l'objet
$id_batiment Id de la définition du bâtiment
x Position x du bâtiment
y Position y du bâtiment
royaume Royaume auquel appartient le bâtiment
hp HP du bâtiment
nom Nom du bâtiment
type Type du bâtiment
rez Pourcentage de HP/MP à la rez ou distance de vision des tours
rechargement Date à laquelle l'arme de siège pourra à nouveau tirer
image Image du bâtiment
date_construction Date de construction du bâtiment
point_victoire Nombre de points de victoire gagnés lorsque le bâtiment est détruit
