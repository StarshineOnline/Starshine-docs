Renvoie une propriété d'un buff / débuff particulier actif sur le personnage ou l'ensemble de ceux-ci.

Paramètres:
$nom Nom (type) du (dé)buff recherché, renvoie tous les buffs actifs si vaut false.
$champ Propriété recherchée (correspond à un champ dans la bdd).
$type Si false on prend le premier buff, si true celui dont le type correspond à $nom.
Renvoie:
Tableau des buffs ou valeur demandée.
