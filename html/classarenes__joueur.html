Description détaillée

Classe de base pour les personnages dans les arènes, liée à la table arenes_joueurs de la base de données.

)

Constructeur.

Le constructeur peut être utilisé de plusieurs façons: -arenes_joueur($event, $id) qui récupère les informations de l'objet dans la base. -arenes_joueur($event, $vals) qui crée l'objet à partir d'information déjà récupèrées sans la base. -arenes_joueur($event, $perso, $statut, $arene, $partie, $x, $y, $groupe, $hp) qui crée une nouvelle arène.

Paramètres:
$event object Objet représentant l'event dont le participant fait parti
$id int Id de l'entrée dans la base.
$vals array Tableau associatif contenant les entrées de la base de données.
$perso int/object objet représentant le personnage ou id de celui-ci.
$statut int statut du personnage par rapport au passage dans l'arène (en attente, en cours, fini).
$arene int/object arène où a lieu la partie.
$partie int/object partie de l'event à laquelle correspond ce passage dans l'arène.
$x int position horizontale du personnage où il doit être téléporté.
$y int position verticale du personnage où il doit être téléporté.
$groupe int/object groupe du personnage.
$hp int hp du personnage après le téléport.
