Description détaillée

Classe de base pour les personnages dans les arènes, liée à la table arenes_joueurs de la base de données.

)

Constructeur.

Le constructeur peut être utilisé de plusieurs façons: -arenes_joueur($event, $id) qui récupère les informations de l'objet dans la base. -arenes_joueur($event, $vals) qui crée l'objet à partir d'information déjà récupèrées sans la base. -arenes_joueur($event, $perso, $statut, $arene, $partie, $x, $y, $groupe, $hp) qui crée une nouvelle arène.

Paramètres
$eventobject Objet représentant l'event dont le participant fait parti
$idint Id de l'entrée dans la base.
$valsarray Tableau associatif contenant les entrées de la base de données.
$persoint/object objet représentant le personnage ou id de celui-ci.
$statutint statut du personnage par rapport au passage dans l'arène (en attente, en cours, fini).
$areneint/object arène où a lieu la partie.
$partieint/object partie de l'event à laquelle correspond ce passage dans l'arène.
$xint position horizontale du personnage où il doit être téléporté.
$yint position verticale du personnage où il doit être téléporté.
$groupeint/object groupe du personnage.
$hpint hp du personnage après le téléport.
