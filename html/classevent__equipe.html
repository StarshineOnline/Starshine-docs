Description détaillée

Classe de base pour les équipes des events, liée à la table event_equipe de la base de données.

Cette classe doit être spécialisée s'il faut enregistrer des données nécessaires, sinon ce n'est pas obligatoire. Elle ne doit pas être crée directement mais par l'intermédiare de l'objet gérant l'event (classe event et dérivées), de manière à créer la bonne classe en cas de spécialisation.

Documentation des constructeurs et destructeur

Constructeur.

Le constructeur peut être utilisé de plusieurs façons: -event_equipe($event, $id) qui récupère les informations de l'objet dans la base. -event_equipe($event, $vals) qui crée l'objet à partir d'information déjà récupèrées sans la base. -event_equipe($event, $nom, $royaume) qui crée une nouvelle équipe.

Paramètres:
$event object Objet représentant l'event dont le participant fait parti
$id int Id de l'entrée dans la base.
$vals array Tableau associatif contenant les entrées de la base de données.
$nom string nom de l'équipe.
$equipe int id du royaume, si l'équipe défend les couleurs d'un royaume, peut-être null.
