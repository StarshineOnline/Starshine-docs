Description détaillée

Classe gérant l'affichage d'un tableau.

Comme pour le code HTML, la création se fait ligne par ligne. Par défaut, la première ligne est définit comme étant l'en-tête et les suivantes des lignes "normales". La première ligne est créée automatiquement lor de la création du tableau.
