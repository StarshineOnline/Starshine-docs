Class PNJ.

Classe PNJ

Classe représentant un PNJ et permettant de l'ajouter, le modifier ou le supprimer de la base.

Documentation des constructeurs et destructeur

)

Constructeur.

Constructeur.

Le constructeur peut être utilisé de plusieurs façons: -PNJ() qui crée un pnj "vide". -PNJ($id) qui récupère les informations du pnj dans la base. -PNJ($nom, $x, $y, $image, $texte) qui crée un nouveau pnj.

Paramètres
$idint Id du pnj dans la base.
$nomString Nom du pnj.
$xint Coordonnée en X du pnj.
$yint Coordonnée en Y du pnj.
$imageString L'adresse? de l'image du pnj.
$texteString
