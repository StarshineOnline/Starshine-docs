Description détaillée

Classe de base pour la gestion de l'interface.

Cette classe fournie les méthode pour créer les autres classes, permettant ainsi d'avoir plusieurs interfaces possibles. Elle correspond à l'interface par défaut, les classes dérivées permettent de définir d'autres interfaces.
