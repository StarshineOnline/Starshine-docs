db::add ( $  obj  ) 

add

Cette fonction permet d'ajouter les valeurs des propriétés d'un objet d'une table correctement définis. La fonction reçoit un argument qui est une class complète. Le nom de cette class est le nom de la table de la base de donnée où travailler. Il doit y'avoir au moins autant de propriété dans la class que de champ dans la table.

Paramètres:
$obj : une classe dont le nom est celui de la table où effectuer l'ajout. Les propriétés de l'objet doivent porter les mêmes noms (en minusucle) que les champs de la table.

Changement des valeurs d'un typ ENUM.

Cette fonction permet de changer les valeurs autorisés dans un type ENUM.

Paramètres:
$table (string) le nom de la table
$col (string) le nom de la colonne de type ENUM
$new_enum [optionnel] (array) ou (string), contient la nouvelle définition du type ENUM dans l'ordre requis, ou si string la seule valeur autorisé. Si vide ou non spécifié, aucun changement ne seras effectuer.
$default [optionnel] (string) la valeur par défaut à utiliser lors d'un ajout. Si vide ou non spécifié, la première option ENUM seras utilisé comme type par défaut
$empty_value [optionnel] (boolean) autorise ou non les valeurs vides dans l'ENUM. True par défaut
Renvoie:
(boolean) true si la mise à jour à réussis, false si elle a échouer ou si rien n'as été modifier.
